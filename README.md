# CRM con Clientes y Notas

Una aplicación web simple y eficiente para la gestión de clientes y sus notas asociadas, desarrollada con tecnologías web básicas.

## 🚀 Demo en Vivo

Puedes probar la aplicación aquí: [CRM App](https://app-c2cyb-fvzkk40gb-luisernestosalas-2775s-projects.vercel.app)

## 📋 Características

- ✅ Gestión completa de clientes (crear, leer, actualizar, eliminar)
- 📝 Sistema de notas asociadas a cada cliente
- 💾 Almacenamiento local en el navegador
- 📱 Interfaz responsive y amigable
- 🔍 Búsqueda y filtrado de clientes
- 🎨 Diseño moderno y minimalista

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura y semántica
- **CSS3** - Estilos y diseño responsive
- **JavaScript (Vanilla)** - Lógica de la aplicación
- **LocalStorage** - Persistencia de datos local
